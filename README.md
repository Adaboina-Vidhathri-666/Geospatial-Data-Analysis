# Geospatial Data Analysis

## Objective
Analyze location-based sales data to identify business expansion opportunities.

## Tools Used
- Python
- Pandas
- VS Code

## Dataset
Sales data from multiple Indian cities.

## Analysis Performed
- Total Sales
- Average Sales
- Highest Sales City
- High Demand but Low Sales Regions
- Exported expansion opportunities

## Output
Generated:
- business_expansion.csv
